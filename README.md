# PasarYangBaru
Halaman web interaktif ini merupakan visualisasi modern dari konsep "Pasar Yang Baru" yang diangkat dari buku Disruption karya Rhenald Kasali, Ph.D. (Guru Besar FEB UI). Materi berfokus pada Bab 2 yang membahas transformasi pasar tradisional menjadi digital marketplace serta fenomena gelombang ketiga internet.
